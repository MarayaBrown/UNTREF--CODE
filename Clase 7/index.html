Multimedia en la Web

El Tag "audio"

El tag audio tiene varios atributos para controles, autoplay, etc.

Una manera de usar el tag audio es la vista arriba, pero también se puede usar de poniendo dentro el contenido source.

cuando lo pongo como source a parte es importante decir el tipo de archivo que estoy poniendo. Importante también poner los controles y algunos otros atributos. Es buena practica poner el preload.

Video


Para el video también tengo la posibilidad de poner distintos atributos como controles, autoplay, preload. Lo puedoponer también en loop. A veces se puede estar puesto como fondo. En ese caso va con autoplay y loop. Por otro lado es buena costumbre poner el video en varios formatos para que si el usuario no puede reproducir el video en un formato el código salte automaticamente a otro. Poster sirve para poner una imagen para poner hasta que el video este cargado para reproducir
